# **Athen**

A **Hypixel SkyBlock Quality-of-Life** mod, developed by **[me](https://github.com/skies-starred)** purely for fun.

---

## Branches

``master`` - main branch for athen development.<br>
``nebulune`` - main branch for nebulune development, check it out [here](https://github.com/skies-starred/Nebulune)

---

## Features

### General

* Colored enchants
* Commission display
* Item tweaks:
    * Show item age
    * Show item price
    * Show item hex color
    * Show stars as stack size
    * Show cake numbers
    * Remove gear score
    * Remove vanilla enchants
* Item quality indicator
* Kat reminder
* Keybinds
* Lag detector
* Mark lobbies
* Missing enchants
* Snap tap
* Sound replacer
* Terminator sounds
* Wardrobe keybinds

### Render

* Arrow hitboxes
* Custom tooltip
* Ichor pool area
* Item name position
* Render optimiser
* Visual words [Text replacer]

### Slayers

* Carry tracker
* Miniboss alerts
* Slayer display
* Slayer highlights:
  * Bosses
  * Minibosses
  * Demons
* Slayer info: displays information about slayers at their feet
* Slayer sounds: mute specific sounds
* Slayer stats
* Slayer timers (spawn & kill)
* Vengeance damage tracker (chat display)
* Vengeance activation timer

### Kuudra

* Auto re-queuer
* Block useless perks
* Build info
* Carry tracker
* Custom titles
* Fresh tool helpers
* Kuudra info
* Rend damage tracker
* Run breakdown
* Run splits
* Stun helper
* Supply waypoints
* Teammate highlights

### Dungeons

* Carry tracker
* Terminal clicks display
* Terminal solver
* Terminal simulator
* Terminal waypoints
* Party finder
  * Player stats on hover
  * Highlighted parties
  * Stats preview on join
* Rare item alerts
* Watcher helper
