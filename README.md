# **Athen**

A **Hypixel SkyBlock Quality-of-Life** mod, developed by **[me](https://github.com/skies-starred)**.

---

## Features

### General

* Colored enchants
* Commission display
* Item tweaks:
  * Show item age
  * Show item hex color
  * Show stars as stack size
  * Show cake numbers
  * Remove gear score
  * Remove vanilla enchants
* Item quality indicator
* Item value: lowest bin, craft cost
* Kat reminder
* Keybinds
* Lag detector
* Mark lobbies
* Missing enchants
* Snap tap
* Sound replacer
* Terminator sounds
* Wardrobe keybinds

### Render

* Arrow hitboxes
* Custom scale
* Custom tooltip
* Ichor pool area
* Item name position
* Item rarity background
* Radial menus
* Render optimiser
* Visual words [Text replacer]

### Slayers

* Carry tracker
* Cocoon alert and timer
* Miniboss alerts
* Slayer display
* Slayer highlights:
  * Bosses
  * Minibosses
  * Demons
* Slayer info: displays information about slayers at their feet
* Slayer sounds: mute specific sounds
* Slayer stats
* Slayer timers (spawn & kill)
* Vengeance damage tracker (chat display)
* Vengeance activation timer

### Kuudra

* Auto re-queuer
* Block useless perks
* Build info
* Carry tracker
* Custom titles
* Fresh tool helpers
* Kuudra info
* Rend damage tracker
* Run breakdown
* Run splits
* Stun helper
* Supply waypoints
* Teammate highlights
* Timers for build start, and supply spawn

### Dungeons

* Carry tracker
* Terminal clicks display
* Terminal solver
* Terminal simulator
* Terminal waypoints
* Party finder
  * Player stats on hover
  * Highlighted parties
  * Stats preview on join
* Rare item alerts
* Watcher helper
